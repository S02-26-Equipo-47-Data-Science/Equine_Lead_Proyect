# EquineLead: Motor de Crecimiento Basado en Datos para la Industria Equina
---
Sistema de crecimiento basado en la implementación de una estrategía dual de modelos de Machine Learning: Random Forest y K-means.

---
Objetivo del proyecto
El proyecto busca responder fundamentalmente dos preguntas:
¿El usuario ha mostrado interés en adquirir un producto o servicio del sitio de e-commerce?
Si el comportamiento del usuario ha evidenciado interés en la oferta del sitio de e-commerce, ¿qué tipo de content hoock se puede aplicar para el cierre de la venta?
Con base en estas dos pautas, el desarrollo del proyecto se centra en entrenar un modelo RandomForest como punta de lanza para comprender el comportamiento y los interéses de los usuarios mediante el uso del historila de cada uno de los usuarios que componene la base de datos del negocio en un periodo de un año. Por otra parte, se aplico un modelo de Aprendizaje no supervisado: K-means, el cual tiene el objetivo de conocer los intereses de los usuarios y clasificarlos en grupos para cerrar la pinza de venta y saber que tipo de producto o servicio ofrecerle, es decir una estrategía de content hooks.
